검색 가이드 닫기

1. 단어를 포함해서 검색하고 싶을 경우 단어 앞에 ‘+’ 를 넣어 주세요.

‘+’ 앞에는 공백을 포함해야 합니다.

예시
지역 +문화
※ 검색 창에 ‘지역 +문화’를 입력 했을 경우 ‘지역’에 대해 ‘문화’가 포함된 데이터가 검색됩니다.

2. 단어를 제외해서 검색하고 싶을 경우 단어 앞에 ‘-’ 를 넣어 주세요.

‘-’ 앞에는 공백을 포함해야 합니다.

예시
날씨 -흐림
※ 검색 창에 ‘날씨 -흐림’를 입력 했을 경우 ‘날씨’에 대해 ‘흐림’을 제외한 데이터가 검색됩니다.

3. 하나 이상 포함해서 검색 하고 싶을 경우 단어 앞에 ‘ㅣ’ 를 넣어 주세요.

‘ㅣ’ 앞/뒤에는 공백을 포함해야 합니다.

예시
스포츠 ㅣ 야구 ㅣ 축구
※ 검색 창에 ‘스포츠 ㅣ 야구 ㅣ 축구’를 입력 했을 경우 ‘스포츠’에 대해 ‘야구’와 '축구’중 하나 이상 포함된 데이터가 검색됩니다.
